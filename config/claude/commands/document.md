# Generate Documentation

Create documentation for:

$ARGUMENTS

## Include
1. **Overview**: Brief description of purpose
2. **Usage**: How to use with examples
3. **API Reference**: Parameters, return values, types
4. **Examples**: Practical code examples
5. **Notes**: Important considerations or caveats

## Format
Use JSDoc/TSDoc for inline documentation:
```typescript
/**
 * Brief description.
 *
 * @param paramName - Parameter description
 * @returns Return value description
 * @throws Error description
 * @example
 * ```ts
 * const result = functionName(arg);
 * ```
 */
```
